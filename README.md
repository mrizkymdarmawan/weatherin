# weatherin 🌤️

hey, this is my first native iOS app — a weather app built while learning Swift from scratch.

## what it does
 
- fetches real-time weather data based on your location
- search any city in the world and get its current weather
- shows temperature, wind speed, and humidity
- 7-day forecast (coming soon)

## how to run it
 
**requirements**
- macOS (any recent version)
- Xcode 16 or later
- iOS 17 simulator or a real iPhone
 
**steps**
 
```bash
# 1. clone the repo
git clone https://github.com/yourusername/weatherin.git
 
# 2. open in xcode
cd weatherin
open weatherin.xcodeproj
 
# 3. select a simulator from the top bar (iPhone 15 or later recommended)
 
# 4. hit the play button (or Cmd + R)
```
 
no API keys, no `.env` setup, no package installation. just open and run.
